### Type of events

In JavaScript, there are two types of events that can occurs. 

  - User Events
  
    - Example
    ```js
         button.addEventListener('click', () => {
          alert('You clicked me');
         });
    ```
  
  - System Events
  
    - Data Events
    
    - Completion Events
    
    - Error Events
  
    ```js
        setTimeout(() => {
          alert("Timer finished!");
        }, 1000);
    ```
